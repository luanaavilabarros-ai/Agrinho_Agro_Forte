# Agro forte, futuro sustentável

Página web completa e responsiva sobre o tema **“Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente”**.

## Tecnologias usadas

- HTML5
- CSS3
- JavaScript puro

Sem frameworks e sem CSS/JS inline.

## Publicação no GitHub Pages

1. Crie um repositório público no GitHub.
2. Envie os arquivos `index.html`, `styles.css` e `script.js` para a branch principal.
3. Vá em **Settings > Pages**.
4. Em **Build and deployment**, selecione:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Salve e aguarde o link do GitHub Pages.

## Topic obrigatório

No GitHub, clique na engrenagem de **About** do repositório e adicione o topic:

```txt
agrinho
```
